## Apiko Agro Mobile
 
 ### Getting started guide
```
$ git clone git@github.com:JSSolutions-Academy/Apiko-Agro-Mobile.git
$ cd ApikoAgroMobile/
```
I\`ve used `yarn` -> [installation guide](https://yarnpkg.com/lang/en/docs/install/) 

You can use `npm` insead, of course.
```
$ yarn
$ yarn run android
```
In separate console:
```
$ yarn start
```
